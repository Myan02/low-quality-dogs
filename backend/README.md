# Low quality dogs backend

To run the backend, make sure you have installed the dependencies in:

```
uv.lock
```

Ensure all proper env variables have been set in a .env file in the same directory as this README.

Check pyproject.toml to make sure your dependencies have the proper versions, if not change the file or update your dependencies.

to start the backend, run:

```
python main.py
```

from the backend directory only.
